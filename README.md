# Seedhemaut Power BI Dashboard

## Overview
This project presents an interactive Power BI dashboard built to analyze Seedhemaut business data. The dashboard provides insights into key performance indicators (KPIs), trends, and overall business performance.

## Objectives
- Analyze business performance
- Track KPIs and metrics
- Identify trends and patterns
- Support data-driven decision making

## Tools Used
- Power BI
- Microsoft Excel / CSV
- DAX
- Power Query

## Dashboard Features
- KPI Cards
- Sales/Performance Analysis
- Trend Visualization
- Interactive Filters and Slicers
- Category-wise Analysis

## Dataset
The dataset contains business-related records used for reporting and visualization purposes.

## Key Insights
- Identified top-performing categories.
- Analyzed monthly performance trends.
- Highlighted areas requiring improvement.
